# JavaScript Objects & Data Handling 📦

This repository contains the third homework assignment for the GoIT Fullstack Developer course. It focuses on mastering **JavaScript Objects**, property manipulation, and modern ES6+ techniques for handling data structures.

🔗 **[Live Demo](https://ibuizle.github.io/goit-js-hw-03/)**

## 📖 About the Project

The core objective of this assignment was to learn how to store, access, and modify complex data. The tasks involve building logic for working with object properties, iterating through objects, and using advanced syntax to combine or copy data efficiently.

### Key Learnings & Features:
- **Object Manipulation:** Adding, updating, and deleting properties dynamically.
- **Object Iteration:** Using `for...in` loops and methods like `Object.keys()`, `Object.values()`, and `Object.entries()`.
- **Destructuring:** Simplifying code by extracting values from objects and arrays into distinct variables.
- **Spread & Rest Operators:** Using `...` to merge objects, copy arrays, and handle a variable number of function arguments.
- **Complex Data Structures:** Working with arrays of objects, which is a standard format for data received from APIs.



## 🛠 Built With

- **HTML5:** Basic page structure.
- **Vanilla JavaScript:** ES6+ features focused on Object logic.
